# Password-Manager (Anleitung)

## Abhängigkeiten
Zur Verwendung des Programms müssen einige packages installiert werden.

`pip install cryptography`  
`pip install pyotp`  

**Ohne diese Pakete funktioniert das Programm nicht!**
